# MyApp project

## How to build

- Build Greeting Project  
```
$ cd Greeting
$ mkdir build
$ cd build
$ cmake ..
$ cmake --build . 
```

- Build Living Project  
```
$ cd Living
$ mkdir build
$ cd build
$ cmake ..
$ cmake --build . 
```

- Build Myapp Project  
```
$ cd MyApp 
$ mkdir build
$ cd build
$ cmake ..
$ cmake --build . 
```

- Execute application

```
$ ./myapp
```
